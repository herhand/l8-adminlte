## Installation Guide
    1)Clone Or Download The Repository

    2)Set the .env File with the relevant configuration

    2)Goto to project root and open a Terminal Enter Following Commands 
        * composer install
        * php artisan key:generate
        * php artisan migrate
        * php artisan db:seed
